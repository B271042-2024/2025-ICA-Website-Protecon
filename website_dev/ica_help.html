Introduction

Prote-Con is developed for protein conservation analysis. It permits sequence alignment, conservation plot, motif search and phylogenetic tree construction.

As a biologist, coding might not be an option. Hence, we offer you solutions. Run few selected tools and download your data.

  1. Providing input. Only FASTA format are permitted in this website. E.g.:

    2. >sequence name

    KDSJBDJBFSDFSBDFKEPJSBNBJVBSKDBFBEK

  2. If the sequences are retrieved from NCBI Protein webpage via Prote-Con, users are allowed to filter them for analysis.

  3. Protein conservation analysis
    • ClustalO (v1.2.4) is used to align protein sequences. Huge gaps within the sequences may indicate the presence of very distant or unrelated sequences hence, it is possibly useful to exclude them from the analysis.
    • EMBOSS patmatmotifs (v6.6.0.0) is used to search for motifs. Protein motifs are short, recurring amino acid sequences that are considered biologically significant and form functional domains.
    • EMBOSS plotcon (v6.6.0.0) is used to plot protein sequence conservation. Highly conserved regions will have high value in the plot, suggesting that the mutation rates within the regions are very low. It may also suggest that the regions may have significant functional constraints that may be useful for evolutionary studies.
    • IQTREE (v2.2.0) is used to construct a phylogenetic tree based on protein sequences. Protein sequences are more conserved than DNA, and often preferred in evolutionary studies, especially when involving a more distant organisms. Those with similar functions usually tend to cluster together in a phylogenetic tree.

How to use it?

Input protein FASTA sequences (restricted to protein sequences)

  1. Method 1: Use your saved Job ID
    • Restricted for returning users who saved their Job ID and created a username.
    • Purpose: Only to re-run analysis and save output files.
    • Instructions: Fill in your username and Job ID. Press Submit.

  2. Method 2: Retrieve sequences from NCBI Protein
    • Use the examples next to the box to test the function.
    • Purpose: To retrieve FASTA sequences from NCBI Protein website.
    • Instructions: Fill in the protein name and organism. Press Submit.

  3. Method 3: Upload FASTA file
    • Restricted to .fasta, .fa and .txt files.
    • Purpose: To upload FASTA file.
    • Instructions: Select FASTA file from your computer. Press Upload.

Preview and filter sequences (only for Method 2 [from NCBI Protein])

  1. Select the relevant rows in the table.
  2. Press Remove.
  3. Press Submit once ready.

Select tools to run analysis

  1. ClustalO (v1.2.4)
    • Objective: To align protein sequences.
    • Mode: Default

  2. EMBOSS patmatmotifs (v6.6.0.0)
    • Objective: To search for motifs from protein sequences.
    • Mode: Default

  3. EMBOSS plotcon (v6.6.0.0)
    • Objective: To plot protein sequence conservation.
    • Mode:
      • Window size (3): 4 for fine details (conservation score based on 4 consecutive bases at a time), 8 for moderate (conservation score based on 8 consecutive bases at a time), 12 for smooth (conservation score based on 12 consecutive bases at a time)
      • Graph (2): png for plot, data for .txt file containing conservation scores

  4. IQTREE (v2.2.0)
    • Objective: To construct phylogenetic tree based on the protein sequences provided.
    • Mode: Bootstrap parameter of 1000.
    • Only suitable for 3+ protein sequences.

Save your session

  1. Objective: To save session and enable users to return to previous data by using the Job ID and username
  2. Instructions: Create a username and press Save. Please keep the Job ID for returning purposes.
  3. Username is not restricted. It is only used for data tracking purposes in case users forget Job ID. Please contact us for support.
